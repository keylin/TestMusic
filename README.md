## 📖 项目介绍 (Introduction)

**音乐歌单助手** 是一个简单、高效的在线工具，专为音乐爱好者设计。它可以将各大主流音乐平台的歌单链接快速解析为标准化的文本列表。

无论你是想迁移歌单、整理音乐收藏，还是通过文本搜索歌曲，这个工具都能帮你一键搞定！我们专注于提供最纯净的“歌名 - 歌手”格式，并内置了智能去重和歌名清洗功能。

🔗 **支持平台**: 网易云音乐 (NetEase Cloud Music) | QQ音乐 (QQ Music) | 汽水音乐 (Qishui/Douyin Music)

---

## ✨ 核心功能 (Features)

- **🎼 多平台全面支持**
  完美支持 **网易云音乐**、**QQ音乐**、**汽水音乐** 的歌单分享链接解析。

- **🧹 智能格式清洗**
  自动移除歌名中的冗余信息（如 `(Live)`, `(伴奏)`, `【官方版】` 等），统一输出干净整洁的 **`歌名 - 歌手`** 格式，提高跨平台匹配成功率。

- **🚫 自动智能去重**
  对于歌单中重复的歌曲（相同的歌名和歌手），程序会自动进行去重处理，同时保留歌单原始的排列顺序。

- **📥 便捷导出选项**
  - **一键复制**: 直接将解析结果复制到剪贴板。
  - **Excel 导出**: 支持将歌单下载为 `.xlsx` 表格文件，方便在 Excel 中管理。

## 🚀 使用指南 (Usage)

1. 打开您的音乐 APP（网易云/QQ音乐/汽水音乐）。
2. 找到您想要导出的歌单，点击 **分享** -> **复制链接**。
3. 将链接粘贴到本工具的输入框中。
4. 点击 **“获取歌单”** 按钮。
5. 等待解析完成后，您可以：
   - 点击 **“一键复制”** 获取纯文本。
   - 点击 **“下载表格”** 保存为 Excel 文件。
